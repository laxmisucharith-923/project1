# project1
Guess The Number Game
This is a simple Python game where the computer generates a random number between 1 and 100, and the player has to guess it.
The game gives hints like:-
Too High
Too Low
It also tracks:
Number of attempts
Time taken
Performance rating
Features
Random number generation
User input validation
Hint system
Attempt counter
Replay option
Technologies Used
Python
random module
time module
